A Netcraft FED Test application

  • * This app was developed using Angular 1.6.4
  • * It uses Twitter as it's database and enables searching for users
  • * The search is an auto complete type although the results are shown on the side
  • * Clicking on a user name will fetch it's latest tweets - you may 'Load More ...'
  • * Developed using jQuery, Bootstrap and Bootswatch (which enables a paper material design)
